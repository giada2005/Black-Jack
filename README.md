# JBlackJack 🃏

JBlackJack è un'applicazione desktop che ricrea il classico gioco di carte Blackjack, sviluppata in **Java** come progetto per l'esame di Programmazione III.

## 🚀 Panoramica del Progetto
Il software simula una partita completa di Blackjack tra un giocatore umano, diversi Bot e un Dealer. Include un sistema di gestione profili, statistiche avanzate e un'interfaccia grafica dinamica.

## 🛠️ Architettura e Tecnologie
* **Linguaggio:** Java
* **Interfaccia Grafica:** Java Swing
* **Architettura:** Model-View-Controller (MVC) per una netta separazione tra logica e UI.
* **Java Stream API:** Utilizzate per la gestione efficiente della collezione di carte e il calcolo dei punteggi.

## 🏗️ Design Pattern Implementati
Il progetto dimostra l'applicazione pratica di diversi pattern software:
* **Strategy Pattern:** Per definire i diversi comportamenti dei Bot e del Dealer.
* **Observer Pattern:** Per aggiornare l'interfaccia grafica in tempo reale al variare del modello dati.
* **Composite Pattern:** Per la struttura gerarchica dei componenti Swing.

## 🎮 Funzionalità Principali
* **Personalizzazione:** Creazione profilo con avatar (Mario/Sonic) e salvataggio statistiche.
* **IA dei Bot:** Logiche di gioco differenziate (es. il Dealer si ferma a 17).
* **Sistema di Progressioni:** Calcolo di esperienza e livelli in base alle vittorie.
* **User Experience:** Animazioni per la distribuzione delle carte e feedback visivo immediato.

## 📥 Come eseguire il progetto
1. Clona il repository: `git clone https://github.com/giada2005/Black-Jack.git`
2. Compila i file sorgente tramite il tuo IDE preferito (IntelliJ, Eclipse, NetBeans) o via terminale.
3. Esegui la classe principale `JBlackJack`.
